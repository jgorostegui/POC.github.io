Este tutorial de CMake proporciona una guía paso a paso que cubre problemas comunes del sistema de compilación que CMake ayuda a resolver. Ver cómo varios temas trabajan juntos en un proyecto de ejemplo puede ser muy útil. La documentación del tutorial y el código fuente de los ejemplos se pueden encontrar en el directorio Ayuda / guía / tutorial del árbol de código fuente de CMake. Cada paso tiene su propio subdirectorio que contiene código que puede usarse como punto de partida. Los ejemplos del tutorial son progresivos, de modo que cada paso proporciona la solución completa para el paso anterior.
